== About this document

This is the Phase 1 handoff of the Socratic Code-Theory Recovery
workflow. Each question below is an `[OPEN]` leaf from
`docs/QUESTION_TREE.adoc` that cannot be answered from the code alone.
Questions are grouped by the role best placed to answer them. Each
question is short enough to need only 1–3 sentences in reply.

When you answer, cite the Q-ID and prefix the response with `(team
answer)` so Phase 2 documentation can trace back to its source.
