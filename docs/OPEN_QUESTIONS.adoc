== About this document

These are the `[OPEN]` leaves of `QUESTION_TREE.adoc` — questions that *cannot*
be answered from the Mistral Vibe code or configuration. This is the
*"Between Phases"* hand-off of the Socratic Code-Theory Recovery workflow.

*Action required:* route each section below to the named role. Each question is
answerable in 1–3 sentences. Once answers are collected, Phase 2 (synthesis of
PRD, Specification, arc42, and ADRs) can proceed — answers will be marked
`(team answer)` and code-derived claims keep their `file:line` citations.

Each entry shows its Q-ID (traceable back to `QUESTION_TREE.adoc`) and Category.
