== 1. Product purpose

Mistral Vibe is an open-source, command-line coding assistant powered by
Mistral's models. It gives a developer a conversational interface to a
codebase, allowing natural-language exploration and modification of a project
through a set of agent tools. (`README.md:20-22`, `pyproject.toml:4`)

It ships under the Apache-2.0 license (`LICENSE`, `pyproject.toml:14`) and is
distributed on PyPI as `mistral-vibe` (`pyproject.toml:2`).

|===

`vibe/cli/entrypoint.py:32-125`.

== 4. Key product constraints

* Runs on Python ≥ 3.12 (`pyproject.toml:21`).
* UNIX environments are officially supported; Windows is best-effort
  (`README.md:24-25`).
* Requires a Mistral API key (or a configured alternative provider) to operate
  (`vibe/core/config/_settings.py:71-78`, `:848-857`).
* Safety-first: tool executions require approval unless an agent profile or
  tool policy authorizes them (`docs/use_cases/UC-003.md` BR-006).
